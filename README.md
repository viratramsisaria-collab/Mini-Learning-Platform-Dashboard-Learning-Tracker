# Mini Learning Platform

A full-stack MERN learning management platform where users can create courses, manage lessons, track learning progress, and maintain their own educational content through a modern web interface.

## 📚 Overview

Mini Learning Platform is a MERN stack application designed to simulate the core functionality of modern e-learning platforms.

Users can create accounts, build courses, organize lessons, and track their learning progress while interacting with a clean and responsive interface.

## ✨ Features

### Authentication

* User Registration
* User Login
* User Logout
* JWT Authentication
* Protected Routes

### Course Management

* Create Courses
* View Courses
* Update Course Information
* Delete Courses
* Course Details Page

### Lesson Management

* Create Lessons
* View Lessons
* Organize Course Content
* Edit Lessons
* Delete Lessons

### Progress Tracking

* Mark Lessons as Completed

* Responsive Design
* Modern UI
* Protected User Actions
* Easy Navigation
